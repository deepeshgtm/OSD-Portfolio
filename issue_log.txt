
 The current implementation sums the values of the two highest pairs and then multiplies by 2, which 
 is incorrect.
Example: If dice are [4,4,3,3,1], the correct score should be (4×2) + (3×2) = 14, but the current code 
returns (4 + 3) × 2 = 14 (coincidentally correct in this case, but fails for other inputs like [5,5,2,2,1] 
where it should be (5×2) + (2×2) = 14 but incorrectly returns (5 + 2) × 2 = 14).
 def DoublePairs(self):
        counts = [self.dice.count(i) for i in range(1, 7)]
        pairs = [(i + 1) for i in range(5, -1, -1) if counts[i] >= 2]
        if len(pairs) >= 2:
            return (pairs[0] + pairs[1]) * 2
        return 0
    
solution
def DoublePairs(self):
    counts = [self.dice.count(i) for i in range(1, 7)]
    pairs = [(i + 1) * 2 for i in range(5, -1, -1) if counts[i] >= 2]  # Multiply each pair by 2 first
    if len(pairs) >= 2:
        return pairs[0] + pairs[1]  # Sum the two highest pairs
    return 0
